# CICD-with-MLOps
Work with CICD Pipeline using Github Action

step by step workflow

1.Create first repoditory in github 'CICD-with-MLOps'.
2.Open command shell use commands
     1. git clone 
     2. cd 
     3. code .   --- to use open VS code.

3.create app.py file

4.'mkdir -p .github/workflows'           ---- use command to create github/workflows folder.  

5.'touch .github/workflows/ci.yml'       ---- to create ci.yml file

6.write code in ci.yml

7.check git status - 'git status'

8.add file in github - 'git add .'

9.Commit Changes   - git commit -m "Added GitHub Actions CI pipeline"

10.Push to GitHub   - git push origin main
